hacks/path_concat

class String < Object

    Allow easy manipulation of paths using String./

    E.g.: 

      prefix = '/usr'/'local'
    #=> "/usr/local"
  prefix/'bin'
    #=> "/usr/local/bin"
  source_prefix/'linux-'+source_version/'.config'
    #=> "/usr/src/linux-2.2.18/.config"
