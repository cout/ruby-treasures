class Module < Object

    A hack to allow specifying public/private/protected using a block. Based on an idea proposed by David Alan Black.

    E.g.: 

    class Thing
  private do
    def hidden
      puts 'hi'
    end
  end

      def visible
    hidden
  end
end

    PUBLIC
    PROTECTED
    PRIVATE
    SECRET

    Public Methods

    private(*args, &block)
